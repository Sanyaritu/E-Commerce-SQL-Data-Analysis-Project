# E-Commerce-SQL-Data-Analysis-Project
This project demonstrates how SQL can be used to analyze real-world e-commerce data and extract meaningful business insights. The database is designed with multiple related tables (customers, products, orders, and order details) to simulate a realistic sales system.

Using SQL queries, the project analyzes revenue trends, customer behavior, product performance, and category-wise sales. It includes both basic and advanced SQL concepts such as joins, aggregations, grouping, filtering, subqueries, and window functions.
